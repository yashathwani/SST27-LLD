Java 17 Design Pattern Assignments (Plain Java)
==============================================

This bundle contains five small, framework-free assignments:
1) Adapter — Payments (Refactoring)
2) Adapter — CSV Import (Feature Implementation)
3) Flyweight — Glyph Styles (Refactoring)
4) Facade — Report Bundle Export (Feature Implementation)
5) Facade (+Adapter optional) — Video Pipeline (Refactoring)

Build & Run (per exercise)
--------------------------
cd <exercise>/src
javac com/example/**/**/*.java
java com.example.<package>.App
